# 🔐 AI-Based Phishing Website Detection System

## 📌 Project Overview
This project detects phishing websites using Machine Learning and real-time feature extraction.

## 🚀 Features
- Real-time URL analysis
- Domain age check
- SSL detection
- DNS verification
- Subdomain & IP detection
- 90%+ accuracy using Random Forest

## 🛠 Tech Stack
- Python
- Flask
- Scikit-learn
- Pandas
- WHOIS
- Bootstrap

## ▶️ How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run application:
python app.py

3. Open browser:
http://127.0.0.1:5000
